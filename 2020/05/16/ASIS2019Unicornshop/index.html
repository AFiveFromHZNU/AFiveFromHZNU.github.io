[ASIS 2019]Unicorn shop

  • utf-8编码安全

题目中给了提示python和unicode

先进入题目

可以看到界面，

我们只要输入item id 和price就可以购买独角兽

但是当我们输入的price超过一位数，即两位数及以上时，就会报错
Only one char(?) allowed!

钱肯定是够的，这里提示只能输入一个字符，那么有没有字符只有一位但是能表示很大的数呢？

这就是utf-8编码的转换安全问题

https://www.compart.com/en/unicode/

去这个网站可以查到10000等数值的Unicode编码的数据，把0x改为%输入即可

0xE2 0x86 0x82

我用的是这一组数据

%E2%86%82
